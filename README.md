This project focuses on predicting aircraft fuel consumption per flight using a machine learning approach. The objective was to estimate how much fuel an aircraft burns on a single flight using basic information such as flight distance, aircraft size, aircraft category, and whether the flight is domestic or international. This type of prediction is useful for understanding airline operating costs, fuel efficiency, and environmental impact.

The dataset used in this project contained aviation route-level data including total fuel burn, number of flights, aircraft class, number of seats, and route distance. Since the dataset provided total fuel burn across multiple flights on a route, a new variable called fuel per flight was created by dividing total fuel burn by the number of flights. The target variable was then log-transformed to reduce skewness and make the regression model perform better.

Before training the model, the dataset was cleaned by removing invalid entries such as negative fuel values, rows where departure and arrival airports were the same, and rows where fuel per flight was zero or extremely small. Categorical variables such as aircraft class were converted into numerical form using one-hot encoding. The main features used for the model were distance, number of seats, domestic/international indicator, and aircraft class.

The dataset was then split into training and testing sets, and a Linear Regression model was trained using scikit-learn. The model was evaluated using R² score, RMSE, and visualization techniques such as actual vs predicted plots and residual plots. The model performed reasonably well and was able to explain most of the variation in fuel consumption per flight, although the residual plots indicated that some non-linear relationships still exist in the data.

Overall, this project demonstrates a complete machine learning workflow including data cleaning, feature engineering, regression modeling, and model evaluation, applied to an aerospace-related problem. It also shows how machine learning can be used to analyze aviation data and estimate aircraft performance metrics such as fuel consumption.
